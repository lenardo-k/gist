CSS Selectors Reference Table

Selector Name Tag Selectivity Description Tag within HTML
Elements <h1>Heading 1</h1> <a> <p> less specific This selector refers to the HTML-elements, the building blocks of the website itself. <h1> <a> <p>
Pseudoclasses a:hover a:active more specific than elements The pseudoclasses are you used to style elements in a special stage, like when you hover the mouse over it or when it is clicked. Mostly used with the anchor tag. <a> (no correspondence in HTML code)
Classes .class1 .article more specific You can group elements in classes that should get the same styling. <h1 class="article">Heading 1</h1>;
IDs #id1 #teaser most specific For specific elements, you give IDs. One ID can only go to one specific element. <h1 id="teaser">
Combinations
multiples selectors nested .class1 h2 specific to those elements within the given class or ID Selectors can be nested, whereas the latter selector is to be found within the first selector. Nested selectors are separated by space only.
multiple selectors unrelated h1, h2, a, p apply to all selectors listed without nesting Unrelated or unnested selectors can be listed and separated by comma.
